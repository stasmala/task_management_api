Вот инструкция по вашему API с примерами запросов и ответов для каждого эндпоинта. Эти примеры помогут вам протестировать и использовать ваше API.

API Документация

1. Регистрация пользователя

POST /register

Описание: Регистрация нового пользователя.

Запрос:

POST /api/register
Content-Type: application/json

Тело запроса:



{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "password"
}

Ответ:

  • Успех (201 Created):
{
    "message": "User registered successfully."
}
  • Ошибка (400 Bad Request):
{
    "message": "The given data was invalid.",
    "errors": {
        "email": ["The email has already been taken."]
    }
}

2. Авторизация пользователя

POST /login

Описание: Авторизация пользователя.

Запрос:

POST /api/login
Content-Type: application/json

Тело запроса:

{
    "email": "john@example.com",
    "password": "password"
}

Ответ:

  • Успех (200 OK):
{
    "token": "your-jwt-token"
}
  • Ошибка (401 Unauthorized):
{
    "message": "These credentials do not match our records."
}

3. Выход из системы

Описание: Выход из системы для аутентифицированных пользователей.

Ответ:
